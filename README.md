## 서비스 소개 영상

**Moitime의 링크 하나로, 서로 가능한 시간을 공유하고, 겹치는 시간을 바로 확인할 수 있습니다.**
- **날짜와 요일**을 기준으로 타임 보드를 만들 수 있습니다.
- 칸을 클릭하거나 드래그해서 가능한 시간을 표시할 수 있습니다.
- 그룹원들이 가능한 시간을 히트맵을 통해 확인할 수 있습니다.
- **구글 캘린더를 연동해 가능한 시간을 자동으로 채울 수 있습니다.**
