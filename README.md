## 서비스를 만든 이유

회의나 스터디 시간을 정할 때 가장 귀찮은 일은 모두가 가능한 시간을 묻고, 겹치는 시간을 직접 확인하는 과정입니다.

when7meet은 링크 하나로 후보 시간을 공유하고, 참여자들이 가능한 시간을 표시하면, 겹치는 시간을 바로 확인할 수 있게 만든 서비스입니다.

## 기능

- 후보 날짜나 반복 요일로 모임 보드를 만들 수 있습니다.
- 참여자는 이름과 임시 비밀번호만으로 응답하고 나중에 수정할 수 있습니다.
- 칸을 클릭하거나 드래그해서 가능한 시간을 빠르게 표시할 수 있습니다.
- 날짜를 누르면 해당 날짜의 모든 시간을 한 번에 선택할 수 있습니다.
- 그룹 전체 가능 시간을 히트맵으로 확인할 수 있습니다.
- 가장 많이 겹치는 후보 시간을 보고 공유 메시지를 바로 복사할 수 있습니다.

## when2meet 개선 사항
- **타임 테이블의 열을 특정 날짜 뿐아니라 요일도 선택할 수 있습니다.**
- **구글 캘린더를 연결해 가능한 시간을 자동으로 채울 수 있습니다.**
